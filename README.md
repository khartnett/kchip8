Chip8 Emulator in Python using pygame

Run with:
```
$ python3 chip8.py <path to ch8 rom file>
```

Keyboard mapping:
```
Keypad             Keyboard
+-+-+-+-+          +-+-+-+-+
|1|2|3|C|          |1|2|3|4|
+-+-+-+-+          +-+-+-+-+
|4|5|6|D|          |Q|W|E|R|
+-+-+-+-+    =>    +-+-+-+-+
|7|8|9|E|          |A|S|D|F|
+-+-+-+-+          +-+-+-+-+
|A|0|B|F|          |Z|X|C|V|
+-+-+-+-+          +-+-+-+-+
```

Available debugging keys:
```
'p'=play/pause
'['=step
'o'=print opcode with description
'i'=print info
'm'=print memory
't'=terminate
```
